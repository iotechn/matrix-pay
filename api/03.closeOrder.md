### 关闭订单(closeOrder)

#### 请求参数

| 参数名         | 类型   | 必填 | 默认值 | 说明                           | 支持平台 |
| -------------- | ------ | ---- | ------ | ------------------------------ | -------- |
| payChannel | enum | Y |      | 支付渠道 | - |
| appid | string | Y |      | 支付平台APPID | - |
| outTradeNo | string | Y |      | 外部商户订单号 | - |

#### 返回值

| 参数名    | 类型   | 必填 | 默认值 | 说明     | 支持平台 |
| --------- | ------ | ---- | ------ | -------- | -------- |
| resultMsg | string | Y    |        | 结果描述 | -        |
|           |        |      |        |          |          |
|           |        |      |        |          |          |
